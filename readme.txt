== Description ==

Light Translator es un plugin ligero diseñado para traducir cadenas de texto específicas en tu sitio web. Ofrece una interfaz intuitiva para gestionar traducciones globales sin sobrecargar tu servidor.

La versión **1.2.0** incluye una nueva pestaña de ajustes con la opción para eliminar las tablas de traducción al desactivar el plugin.

== Installation ==

1. Sube el plugin a la carpeta `wp-content/plugins/` o instálalo desde el panel de administración de WordPress.
2. Activa el plugin desde el menú de "Plugins".
3. Agrega traducciones globales desde la página "Traducciones" en el menú de administración.
4. Ajusta configuraciones en la pestaña "Ajustes" si lo deseas.

== Frequently Asked Questions ==

= ¿Qué hace este plugin? =
Permite reemplazar textos específicos en tu sitio web con traducciones personalizadas, afectando todo el sitio globalmente.

= ¿Cómo puedo eliminar las tablas de traducción de la base de datos al desactivar el plugin? =
1. Ve a la pestaña "Ajustes" en el menú de administración.
2. Activa la opción para eliminar las tablas al desactivar el plugin.
3. Desactiva el plugin desde el menú de "Plugins" y las tablas serán eliminadas automáticamente.
